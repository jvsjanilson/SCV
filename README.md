# SCV - Sistema de Controlador de Viagens

## Objetivo
O objetivo deste trabalho é aplicar os conceitos de programação orientada a objetos em C++ para desenvolver um sistema completo de controle de viagens, consolidando o aprendizado adquirido ao longo do curso.

## Sistema operacional utilizado

(Linux Mint 22.3 - Cinnamon 64-bit)

## Compilacao do projeto

No diretorio src do projeto execute o seguinte comando no terminal.

``` g++ -std=c++17 -g -Wno-unused-variable -Wall -o sistema_viagem main.cpp Cidade.cpp Trajeto.cpp Transporte.cpp Passageiro.cpp Viagem.cpp ControladorDeTransito.cpp Persistencia.cpp ```
